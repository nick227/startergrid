# US ZIP centroid lookup data

Generated by `npm run geo:download-zips`.

- **Records:** 33,791 five-digit ZCTA centroids
- **Source:** 2022 US Census ZCTA gazetteer coordinates (CSV mirror)
- **License:** US government public domain (Census Bureau ZCTA data)

Do not edit `us-zip-centroids.json` by hand. Re-run the download script to refresh.
