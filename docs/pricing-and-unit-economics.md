# Pricing And Unit Economics

## Pricing Hypothesis

The product is support-heavy at first. Price for setup labor.

Recommended first package:

```txt
$799 setup
$249/month
$199 per assisted channel
```

## Setup Economics

Assumption:

```txt
3 to 6 hours per dealer setup
```

At `$799`:

```txt
3 hours = $266/hour gross before tooling/support
6 hours = $133/hour gross before tooling/support
10 hours = $80/hour gross and too low if repeated
```

Rule:

If average setup exceeds 6 hours, raise setup price or reduce included work.

## Monthly Economics

At `$249/month`:

```txt
100 dealers = $24,900 MRR
250 dealers = $62,250 MRR
500 dealers = $124,500 MRR
```

Support cap target:

```txt
under 30 minutes/dealer/month on base plan
```

If support averages 2 hours/month, `$249` is too cheap.

## Assisted Channel Economics

At `$199/channel`:

```txt
50 dealers x 2 channels = $19,900
100 dealers x 2 channels = $39,800
```

Use this for:

- CarGurus packet
- Cars.com packet
- Autotrader/Cox packet
- TrueCar packet
- partner rep follow-up

Do not bury this labor inside the monthly fee.

## Managed Ads Add-On

Only after storefront and feeds pass.

Starter:

```txt
$499/month management
dealer pays ad spend
```

Back of napkin:

```txt
50 dealers = $24,950 MRR
100 dealers = $49,900 MRR
```

Risk:

Bad inventory data makes ad performance look bad. Require readiness before ads.

## Early Revenue Scenario

```txt
50 dealers onboarded
50 x $799 setup = $39,950
50 x $249 MRR = $12,450 MRR
20 dealers x 2 channels x $199 = $7,960
```

First-cycle revenue:

```txt
$47,910 cash collected
$12,450 recurring base
```

## Pricing Decision Rules

- If dealer needs done-for-you cleanup, charge setup.
- If partner workflow needs manual follow-up, charge per channel.
- If dealer wants ad management, require active storefront and clean feeds first.
- If dealer will not provide documents/photos, do not discount.
- If support demand is high, package managed service instead of SaaS.
