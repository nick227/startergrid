# Dealer Onboarding Checklist

## Required Dealer Data

- legal business name
- DBA
- dealer license number
- rooftop street/city/state/ZIP
- primary contact name/title/email/phone
- website/domain
- inventory size
- desired channels

## Required Documents

- dealer license PDF
- W-9
- certificate of insurance if available
- logo if storefront branding is included
- any marketplace account IDs already owned

## Required Inventory Data

Each vehicle needs:

- VIN
- stock number
- year/make/model
- trim
- mileage
- price
- condition
- exterior color
- body style
- fuel type
- drivetrain
- transmission
- public listing URL or generated storefront URL

## Required Media

Minimum:

- 4 photos per vehicle for pristine/demo quality
- public HTTPS image URLs
- front 3/4
- rear 3/4
- interior
- dashboard or feature/detail

Reject or flag:

- broken URLs
- tiny images
- stock photos posing as vehicle photos
- missing first image
- duplicate image URL across vehicles

## Platform Setup Questions

Ask before activation:

- Does the dealer already have Google Merchant Center?
- Does the dealer have Meta Business Manager?
- Does the dealer have TikTok Ads Manager?
- Does the dealer have Microsoft Advertising?
- Does the dealer have eBay seller access?
- Which marketplaces are already active?
- Which reps or account contacts exist?

## Launch Sequence

1. Dealer profile intake
2. Document intake
3. Inventory import
4. Media validation
5. Storefront generation
6. Baseline readiness
7. Strict readiness
8. Feed/packet generation
9. Assisted channel routing
10. Lead capture test

## Dealer-Facing Deliverable

Give the dealer:

- storefront link
- readiness report
- open blocker list
- generated feed/packet list
- submitted/routed channel list
- next-action checklist
