# Segment: Growing Independent Dealer With Existing DMS

## Buyer

Independent dealer already using DealerCenter, AutoManager, Frazer, vAuto, HomeNet, or another inventory/DMS workflow.

## Real Pain

- DMS exports exist but external activation is still unclear
- marketplace status lives in separate portals/emails
- data quality issues are discovered after submission
- dealer does not know which fields block which platforms
- vendor stack is growing but no neutral launch dashboard exists

## Best Hook

```txt
Keep your DMS. We turn its dealer and inventory output into a platform readiness dashboard, proof folder, and activation workflow.
```

## First Offer

```txt
Launch Audit
$199-$299
```

Then:

```txt
Readiness + Activation Layer
$799 setup + $249-$399/month
```

## What To Emphasize

- coexistence
- export/feed intake
- readiness scoring
- proof folder
- partner status tracking
- no DMS replacement threat

## Discovery Questions

- Which DMS/inventory tool is source of truth?
- Can it export CSV/XML/API?
- Which sites already receive your inventory?
- Who manages marketplace contracts?
- Where do feed errors show up today?
- How do you know a sold vehicle disappeared everywhere?

## Objections

"Our DMS syndicates inventory."

```txt
That helps. We are not claiming exports do not exist. We validate whether the dealer, inventory, media, and partner status are actually ready channel by channel.
```

"We already pay vendors."

```txt
This gives you a neutral proof layer across those vendors: what is ready, what is blocked, what was generated, and what was submitted.
```

## Disqualify If

- dealer has an agency fully managing all activation and reporting
- no access to export data
- unwilling to let Launchpad inspect inventory quality
