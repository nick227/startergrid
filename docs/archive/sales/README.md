| Managed Catalog Ads Setup | clean inventory + ad appetite | `$499/mo+` |

## Qualification Rule

Good prospect:

- has inventory
- wants online visibility
- can provide docs/photos/data
- is confused by platform setup
- will pay for hands-on launch work

Bad prospect:

- wants guaranteed marketplace approval
- will not provide dealer docs
- has no usable vehicle photos
- wants a full DMS/F&I system first
- expects unlimited support at low SaaS pricing
