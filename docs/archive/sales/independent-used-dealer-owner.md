# Segment: Independent Used Dealer Owner

## Buyer

Owner or GM of a 10-100 car used lot.

Usually cares about:

- more leads
- cars showing correctly online
- less vendor confusion
- not paying enterprise software prices

## Real Pain

- inventory listed inconsistently across sites
- weak or stale website
- no one knows why Google/Meta/marketplace feeds fail
- photos exist but are not organized as clean public URLs
- dealer license/W-9/contact info gets requested repeatedly
- platform reps and portals are hard to track

## Best Hook

```txt
We get your inventory online first, then show exactly which platforms are ready, blocked, or waiting on partner approval.
```

## First Offer

```txt
Storefront + Readiness Setup
$799 setup + $249/month
```

Includes:

- dealer profile cleanup
- hosted inventory storefront
- readiness report
- generated feed/packet artifacts
- platform status tracker
- proof folder

## Demo Path

1. Show their dealer profile completeness.
2. Show 3 sample vehicles with photo/readiness checks.
3. Show the storefront listing.
4. Show Google/Meta/TikTok/Microsoft as feedable.
5. Show Cars.com/Autotrader/CarGurus/TrueCar as assisted.
6. Show next action for one blocked channel.

## Discovery Questions

- Where is your inventory online today?
- Who updates sold units?
- Do you know which channels are actually active?
- Do you have public photo URLs?
- What happened last time you tried Cars.com, Autotrader, or CarGurus?
- How many leads came from your website last month?

## Objections

"I already have a website."

```txt
Good. We are checking whether it is usable as a clean inventory destination for feeds, ads, and lead capture. If it is, we use it. If not, we launch a controlled storefront.
```

"Can you put me on every marketplace?"

```txt
We can prepare and track every channel. Some channels allow feed/API work, and some require partner approval or a rep. We show the difference clearly.
```

## Disqualify If

- no inventory list
- no dealer license
- no willingness to fix photos/data
- wants guaranteed lead volume before launch basics
