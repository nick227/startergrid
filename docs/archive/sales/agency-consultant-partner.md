# Segment: Automotive Marketing Consultant Or Agency

## Buyer

Small agency, freelancer, or consultant serving independent dealers.

## Real Pain

- dealer data is messy before campaigns begin
- ads underperform because feeds/photos/landing pages are weak
- agency spends time chasing dealer docs and inventory fixes
- assisted marketplace handoffs are hard to prove
- client asks "what are you doing?" without a tangible proof folder

## Best Hook

```txt
Use Launchpad as the readiness and proof layer before you run ads or hand off partner activations.
```

## First Offer

```txt
Agency workspace or white-labeled proof folder
per dealer setup + monthly platform fee
```

Early pricing:

```txt
$499-$799 setup per dealer
$149-$249/month per dealer
```

## What To Emphasize

- pre-ad data QA
- storefront landing pages
- feed artifacts
- proof folder
- client-facing status dashboard
- assisted marketplace packet history

## Discovery Questions

- How many dealer clients?
- How do you collect inventory/photos today?
- Which ad channels do you manage?
- Do you manage marketplace onboarding?
- How often do client feeds break?
- What do you show clients as proof of work?

## Objections

"We already build landing pages."

```txt
Great. Launchpad is useful if you need readiness checks, platform artifacts, and status proof around those pages.
```

"Can this be white-labeled?"

```txt
That should be an agency-tier feature after the core proof folder and status dashboard are stable.
```

## Disqualify If

- wants only ad campaign management
- does not control or influence dealer inventory quality
- refuses per-dealer setup economics
