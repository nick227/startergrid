# Segment: Small Dealer Group

## Buyer

Operator with 2-10 independent rooftops or multiple brands/lots.

## Real Pain

- each rooftop has different website/listing quality
- inconsistent dealer profile data
- inconsistent photos and inventory fields
- no single view of channel readiness by rooftop
- partner approvals and feeds vary by location

## Best Hook

```txt
We standardize launch readiness across rooftops and show which locations are active, blocked, or waiting on partner action.
```

## First Offer

```txt
Pilot 1 rooftop
$1,500 setup + $399/month
```

Then expand:

```txt
per-rooftop setup + group dashboard
```

## What To Emphasize

- standardization
- multi-rooftop profile completeness
- shared platform registry
- per-location proof folders
- location-level status
- lead capture by rooftop

## Discovery Questions

- How many rooftops?
- Do all rooftops use the same DMS?
- Are marketplace contracts per rooftop or group?
- Do vehicles move between locations?
- Is media handled centrally?
- Who owns vendor relationships?

## Objections

"We need this for all stores."

```txt
Start with the messiest or most urgent rooftop. Use the first proof folder to define the rollout standard.
```

"We need group reporting."

```txt
That is the expansion path. The first milestone is location-level readiness and repeatable launch artifacts.
```

## Disqualify If

- expects enterprise DMS replacement
- wants OEM/franchise compliance workflows first
- no central owner for rollout
